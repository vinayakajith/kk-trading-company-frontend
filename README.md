# KK Trading Company - Frontend

![KK Trading Company](images/farmer_hands_rice.png)

> **Kerala Rice & Spices, Direct from the Land We Farm.**
> A modern, highly-responsive frontend web application designed for KK Trading Company, a legacy business specializing in premium rice and spice trade in Kerala, India since 1963.

This project is the official frontend landing page for KK Trading Company. It serves to establish a strong digital presence, communicate the brand's rich 150-year agricultural heritage, and showcase its primary product offerings (premium rice varieties and spices). The design is focused on emotional grounding, high visual contrast, and a direct, authoritative tone suitable for B2B wholesale trade.

## âœ¨ Features

- **Modern & Responsive Design**: Fully responsive layout optimized for all screen sizes (mobile, tablet, desktop).
- **Heritage Narrative Arc**: Engaging storytelling sections that transition from farming origins to modern-day trading.
- **Dynamic CSS Architecture**: Built using a robust CSS variable system (custom properties) for easy theming and maintainability.
- **Micro-Animations**: Subtle `reveal-up` animations tied to scroll events for a premium, dynamic feel.
- **Partner Focused**: Dedicated section tailored towards target B2B buyers (Retail Chains, Wholesalers, Distributors, Bulk Buyers).
- **Direct Mail Integration**: Simplified inquiry flow using direct `mailto:` integration.

- **HTML5**: Semantic, accessible markup.
